## 1 上游决定因素（暴露 → 炎症）
&emsp;&emsp;定位：充斥着低质量的研究。每年有成千上万篇论文计算各种可能疾病的 NLR/SII，但由于没有标准化检测或控制急性感染，这些论文缺乏临床价值。  
&emsp;&emsp;***科学问题：哪些群体特征、环境因素或行为驱动慢性炎症？这里的目标是初级预防 ——找到可调整的靶点，在炎症开始前阻止它。***
### 1.1 探索生物标志物（持续寻找“炎症 BMI”）：仍然缺乏一个金标准指标。
- 方法脉络：研究人员不再仅用 hs-CRP 或 IL-6，而是使用全血细胞计数（CBC）衍生指标，如中性粒细胞与淋巴细胞比值（NLR）、系统免疫炎症指数（SII）和 SIRI 指标。这些药物目前非常受欢迎，因为它们成本低廉，在人群数据中普遍可得（如 NHANES），并且能平衡先天免疫与适应性免疫。
- 推荐文献：可以参见nature综述
### 1.2 观测性关联研究（Observational Associations）
- 方法脉络：横断面或基线纵向数据。
- 推荐文献：空气污染（PM2.5）、超加工饮食或慢性心理社会压力与 NLR 或 CRP 升高相关。
### 1.3 高级因果关系（组学/暴露体）：
- 方法：表观全基因组关联研究（EWAS）或代谢组学，以精确观察 暴露（如吸烟）如何改变基因甲基化，从而产生炎症状态。

## 3 动力学、阈值与路径（“黑匣子”）
&emsp;&emsp;定位：当前公共卫生流行病学中的“甜蜜点”。研究人员利用大量队列（如英国生物样本库）的重复测量数据，来判断谁在其一生中会失控炎症， 原因如何， 以及 如果我们中断炎症通路，可以消除多少疾病负担。  
***科学问题： 炎症在不同时间和人群中表现如何？在什么具体水平下变得危险？它如何作为暴露与结果之间的桥梁？***
### 3.1 识别干预阈值（剂量-反应关系）：（xxxx年）
- 图表：X 轴=炎症标志物水平;Y 轴=疾病风险（HR/OR）。
- 目标： 找到“拐点”。这种关系是线性的，还是 J 型的？在什么具体的 NLR 值下，死亡风险会突然飙升？
- 方法脉络： 限制三次样条（RCS） 或嵌入 Cox 回归中的分数多项式。
### 3.2 识别高风险人群（生命周期轨迹）：
- 图表：X 轴=时间/年龄;Y 轴 = 炎症标志物水平。
- 目标：观察种群随时间的分歧。例如，区分“快速增加的炎症”组和“持续低”的组，然后观察基线特征（如贫困、肥胖）是否能预测谁最终进入高风险轨迹。
- 方法脉络：潜在阶级增长分析（LCGA） 或基于群体的轨迹建模（GBTM）。
### 3.3 评估路径（调解）：
- 目标：回答：“暴露 X 是否通过慢性炎症途径具体导致结果 Y ？”（例如，肥胖是否会导致抑郁 因为肥胖会导致炎症？）。
- 方法脉络： 介导分析或结构方程建模（SEM）。
  
&emsp;&emsp;*p.s.* 对于前几年比较火热的SEM，本质起源于社会学和心理测量学。应用于横断面自然科学数据时，其因果解释力几乎为零（它只是一个花哨的第一阶段关联）。SEM 仅在应用于纵向数据（如交叉滞后面板模型）时才严格有效，其中暴露在时间 1，炎症在时间 2，结局在时间 3。许多研究者滥用横断面扫描电子显微镜，这是文献中的一个主要局限。
